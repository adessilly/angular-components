 SwitchView

 SwitchView

La SwitchView est similaire à une box, mais permet d'ouvrir un formulaire en popup qui sera synchronisé avec le contenu de la switchView après sa validation

Arguments :
  • title : titre de la box
  • boxOpen : indique si la box est ouverte lors de sa création
  • boxStyle : indique la classe a utilisé pour la box (box-primary, box-info, box-warning, box-danger,...)
  • data : objet de donnée
  • formComponentType : type de formulaire à ouvrir en popup, c'est donc la classe qui contient le formulaire avec les fields etc
  • modeBandeau : indique si on ouvre la box en mode bandeau (avec accès au formulaire en popup) ou directement en mode formulaire
  • initCloneFunction : fonction appelée pour cloner l'objet envoyé à la popup
  • commitCloneFunction : fonction appelée pour mettre à jour l'objet original avec les données du clone
